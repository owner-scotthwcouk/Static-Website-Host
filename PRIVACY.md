# Privacy Policy for Static Website Host

## 1. Data Collection
Static Website Host does not collect, store, or share any personal user data. 

## 2. Dropbox Access
The app only accesses the specific folder created for it within your Dropbox (`Apps/Static Website Host`). It reads Markdown files to convert them into HTML. It does not access any other files or folders in your Dropbox account.

## 3. Data Storage
All processing happens locally on the machine where the script is running. The app does not transmit your files to any third-party servers.

## 4. Contact
For any questions regarding this privacy policy, please open an issue on the GitHub repository.
